# 🏅 Olympics Exploratory Data Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on historical Olympic Games data using Python. The goal is to analyze athlete participation, medal distribution, country performance, gender trends, and other Olympic statistics through data cleaning, analysis, and visualization.

---

## 🎯 Objectives

- Analyze historical Olympic athlete data.
- Clean and preprocess the dataset.
- Explore participation trends across countries and years.
- Analyze medal distribution by country and gender.
- Study athlete demographics such as age, height, and weight.
- Visualize insights using different charts.

---

## 📂 Dataset

The project uses two datasets:

- **Olympics.csv** – Historical Olympic athlete records
- **noc_regions.csv** – National Olympic Committee (NOC) region mapping

After loading the datasets, they are merged using the **NOC** column.

## 📌 Key Insights

- The United States has the highest Olympic participation.
- Gold medals are distributed among several leading countries, with the USA ranking first.
- Female participation has increased significantly over time.
- Summer Olympics include more sports than Winter Olympics.
- Athlete age, height, and weight show noticeable variation across different sports.
